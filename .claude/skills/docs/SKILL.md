# ドキュメント作成ガイド

## 避けるべき表現

AI 特有の文体を避ける。

- `**` による強調
- 絵文字
- 「素晴らしい」「完璧」などの過剰な肯定
- 「基本的に」「本質的に」「実際に」などの不要な修飾
- 「さらに」「また」「加えて」の多用
- 「〜することが重要です」「〜に注意してください」の乱用
- 文末の要約や繰り返し
- 箇条書きの各項目が同じ語で始まる構成

## 心がけること

- 事実を端的に書く
- 必要な情報だけを書く
- 読み手が知りたいことを先に書く
- 一文を短くする
- 具体例があれば示す

## Markdown の使い方

見出しは `##` から始める（`#` はタイトル用）。
コードは言語を指定する。

```typescript
const x = 1;
```

強調が必要な場合はバッククォートで囲む（`このように`）。
